Block Elements

  • div tag
  • section tag
  • p tag
  • list tag
  • heading tag
  • blockquote
  • table
  • forms

Block level elements create a block-level box on the webpage

They typically start on a new line and take up the full available width of the parent container

Block-level elements are often used to create major structural components of a webpage

These are headings, paragraphs, divs, lists and sections

Inline elements

  • span tag
  • a tag
  • strong and em tag
  • img tag
  • br tag
  • span with css tag
  • code tag

Inline elements do not create a new block-level box

they flow within the content and do not flow out of the line

they are used to style or wrap smaller parts of the content
