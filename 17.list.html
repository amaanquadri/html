Types of Lists(description)
ordered list
Here the list is arranged in ascending order numerically
unordered list
Here the list is arranged in random order
description list
here the description of the list is provided along with dt and dd tag

Unordered list:

Unordered list is an type of list where order doesnt matter.
the list point are represented with bullet points.

Ordered list:

Ordered list is a list of items that are sequentially numbered.
order of items matters
items are numbered automatically in ascending order.

Description list:

Description list consist of a list of terms and their corresponding descriptions.
dt tag for the term
dd tag for the description data of the list
