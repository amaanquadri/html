Tables In HTML

HTML tables are used to display data in a tabular format

HTML tables are made up of columns and rows,each cell in table can contain text, images or other elements

HTML tables are defined using tabletag.The table tag is placed in the body of html document

HTML tables has a no of attributes that can be used to control apperance of the table such as width, height and border

tabletag contain tr th td tags

tr tag contain table rows. th tag contain table header cell

td tag defines data cell in table
